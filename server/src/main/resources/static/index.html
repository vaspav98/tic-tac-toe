Скачать игру

ОС Файл для скачивания Инструкция
Linux tic-tac-toe.jar
  1. Убедитесь, что на вашем компьютере установлен JRE версии 8
  2. Открыв терминал, перейдите в директорию, где находится jar файл
  3. Запускаем jar файл:
    • Если java версии 8 стоит по умолчанию, то просто java -jar tic-tac-toe.jar
    • Инача - (путь/до/исполняемого/файла/java/8)/java -jar tic-tac-toe.jar
Windows tic-tac-toe.7z
  1. Распакуйте архив
  2. Найдите в папке файл tic-tac-toe.exe и дважды кликните по нему, чтобы запустить игру
